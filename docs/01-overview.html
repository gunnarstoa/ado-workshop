Workshop Overview

What this workshop is, who it's for, and how to use it
Getting Started ⏱ 15 min read

What Is This Workshop?

This workshop fills a critical gap that the Anaplan ADO Academy course does not cover. The Academy course teaches you how to build ADO — pipelines, links, transformation views, and source data. It does that well. This workshop picks up where the Academy leaves off and covers two topics that every delivery partner needs during real-world implementations:

1

Automate ADO Jobs

Use Anaplan Workflow to schedule ADO runs automatically — no manual triggering required. Configure separate workflow templates per dataset type and set up recurring schedules.

~2 hours
2

Productionize ADO

Walk through the 6-step process to promote your ADO setup from a development environment through UAT to a full production environment with separate models, source data, and links.

~2 hours
Important

This workshop does not teach you how to build ADO from scratch. If you haven't completed the Anaplan ADO Academy course, do that first — it's a prerequisite. This workshop assumes you already know how ADO works and have working ADO jobs from that course.

Who Is This For?

This workshop is designed for:

  • Model Builders who have built ADO pipelines during the Academy course and now need to move them to production
  • Solution Architects who need to understand the full ADO delivery workflow to design and oversee implementations
  • Partners who are working on an Anaplan Apps opportunity and will need ADO for data integration

Case Study: Meridian Corp

All lab exercises use a fictional company called Meridian Corp — a mid-sized manufacturing and distribution company with approximately 2,400 employees across 12 regional offices. Meridian Corp uses Anaplan for financial planning and headcount management, with data flowing from an HRIS and an ERP system into Anaplan via ADO.

HR Data Flow

  • Source: HRIS (exported as CSV)
  • Dataset: Employee master + headcount details
  • Target: HR Model in Anaplan
  • Schedule: Nightly (runs at 11 PM UTC)

Financial Data Flow

  • Source: ERP (exported as CSV)
  • Dataset: Product master + cost data
  • Target: FPA Model in Anaplan
  • Schedule: Weekly (Sundays at 6 AM UTC)

How to Use This Guide

This is a self-paced guide. Work through it in order — the reading pages set up context for each lab. Here's the recommended approach:

1

Read first

Each topic has a reading page that explains the concept and shows the steps with screenshots. Read it fully before starting the lab.

2

Do the lab

Labs are structured as mini-labs — short, focused exercises that build on each other. Each mini-lab has clear objectives and step-by-step instructions.

3

Check your work

Each lab ends with a Sample Answers section showing what the expected outcome looks like. Compare yours to validate your work.

4

Use Reference pages as needed

If something isn't working, check the Troubleshooting page. If you have conceptual questions, check the Q&A page.
